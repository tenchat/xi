## 二、核心知识考点

### （一）高频考点（问答/简答题方向，老师明确划重点，必须熟记并能展开论述）

1. **习近平新时代中国特色社会主义思想创立的时代背景**（五个方面）——导论第一重点
2. **"两个确立"的决定性意义**——导论第二重点
3. **新时代的内涵**：中国特色社会主义进入新时代，是我国发展新的历史方位——第一章唯一考点
4. **中国创造人类文明新形态的三个方面**：①提供全新现代化路径，超越西方现代化模式；②为发展中国家提供新的选择；③需牢牢把握推进中国式现代化的重大原则——第二章
5. **推进中国式现代化必须牢牢把握的重大原则**（五个方面，与"四项基本原则"结合记忆）：
   - 坚持和加强党的全面领导
   - 坚持中国特色社会主义道路
   - 坚持以人民为中心的发展思想
   - 坚持深化改革开放
   - 坚持发扬斗争精神
6. **中国最大的国情是中国共产党的领导**——第三章第一考点
7. **健全党的全面领导制度**——第三章第二考点
8. **人民立场是中国共产党的根本政治立场**——第四章第一考点
9. **促进全体人民共同富裕取得更为明显的实质性进展**——第四章第二考点
10. **新时代全面深化改革开放是一场深刻革命（及其原因）**——第五章第一考点
11. **全面深化改革总目标**：完善和发展中国特色社会主义制度，推进国家治理体系和治理能力现代化——第五章第二考点
12. **全面深化改革开放要坚持正确的方法论**——第五章第三考点
13. **新发展理念的科学内涵与实践要求**（创新、协调、绿色、开放、共享）——第六章
14. **科技强则国家强**（科技是第一生产力、人才是第一资源、创新是第一动力）——第七章
15. **人民民主是社会主义的生命**——第八章第一考点
16. **坚定不移走中国特色社会主义政治发展道路**——第八章第二考点
17. **文化繁荣兴盛是实现中华民族伟大复兴的必然要求**——第十章唯一考点
18. **完善分配制度，推动高质量发展、促进社会公平**——第十一章考点
19. **生态文明建设摆在全局工作的突出位置**——第十二章考点

> 以上19条是老师在课堂上逐章点出的"问答题方向"，是**主观题最可能直接命题**的内容，复习优先级最高。

### （二）基础考点（客观题高频考查，需准确记忆细节）

- **导论**：
  - "两个结合"：马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合（一个是"魂脉"，一个是"根脉"）
  - 思想体系构成：十个明确、十四个坚持（也称"十四个方略"）、十三个方面成就、六个必须坚持
    - 十个明确：集中体现了这一思想的主要观点和基本主张
    - 十四个坚持：构成了新思想的基本方略
    - 六个必须坚持：从世界观和方法论角度，彰显了这一思想的理论品格与鲜明特征
  - 历史地位：当代中国马克思主义、21世纪马克思主义；中华文化和中国精神的时代精华；实现马克思主义中国化新的飞跃（**只说"新的飞跃"，没有"第三次飞跃"的提法**）
- **第一章**：
  - 道路问题是关系党的事业兴衰成败的第一位的问题
  - 一个国家实行什么主义，关键要看这个主义能否解决这个国家面临的历史性课题
  - 中国特色社会主义是党和人民历经千辛万苦、付出巨大代价取得的根本成就，是实现中华民族伟大复兴的必由之路
  - "两个奇迹"：经济快速发展奇迹和社会长期稳定奇迹
  - 中国特色社会主义是科学社会主义理论逻辑和中国社会发展历史逻辑的辩证统一
  - 科学社会主义基本原则：首次在《共产党宣言》中全面阐述，在《资本论》及其手稿等著作中充分展开并深入论证，在《哥达纲领批判》《反杜林论》中进一步阐发
  - 四个自信中，文化自信是更基础、更广泛、更深沉的力量
  - "一个变与两个没有变"：社会主要矛盾已经变化，但我国仍是世界最大发展中国家、仍处于社会主义初级阶段的基本国情没有变
  - 新时代的伟大变革，在党史、新中国史、改革开放史、社会主义发展史、中华民族发展史上都具有里程碑意义
  - 党的基本理论、基本路线、基本方略；基本路线是国家的生命线、人民的幸福线，是社会主义初级阶段的基本路线，可概括为"一个中心、两个基本点"
  - 中国梦的内涵：国家富强、民族振兴、人民幸福，三个维度辩证统一
  - "五位一体"总体布局与"四个全面"战略布局
- **第二章**：
  - "五个必由之路"之一——团结奋斗是中国共产党、中国人民最显著的精神标识，也是中国人民创造历史伟业的必由之路
  - 推进中国式现代化要处理好"六对重大关系"
- **第三章**：
  - 中国共产党的自身优势是中国特色社会主义制度优势的主要来源
  - 党的自身优势：政治领导力、思想引领力、群众组织力、社会号召力（其中政治领导力是马克思主义政党第一位的能力）
  - 党的领导是全面的、系统的、整体的，但不意味着事事都要由党具体包办
  - 党中央集中统一领导是党的领导的最高原则；维护党中央权威和集中统一领导，是马克思主义执政党成熟的重大建党原则
- **第四章**：
  - 民心是最大的政治，决定事业兴衰成败
  - 群众路线是党始终坚守的根本工作方法，调查研究是重要工作方法
  - 共同富裕是中国特色社会主义的本质要求，是中国式现代化的重要特征，要坚持有底线、渐进式推进
- **第六章**：
  - 主要内容为习近平经济思想
  - 社会主义基本经济制度新概括：从所有制、分配制度、资源配置体制机制角度展开，"两个毫不动摇"是所有制基础
  - 新发展格局：以国内大循环为主体、国内国际双循环相互促进
- **第八章**：
  - 马克思主义认为，民主首先是、而且主要是一种国家制度
  - 全过程人民民主体现"四个相统一"
- **第十章**：
  - 中国特色社会主义文化的三个主要组成部分
  - 意识形态关乎旗帜、关乎道路、关乎国家安全
  - 弘扬伟大建党精神，理解其具体内涵
- **第十一章**：
  - "让人民生活幸福是国之大者"
  - 加快建设市域社会治理现代化：市域是将风险矛盾化解在基层、解决在萌芽状态最直接、最有效率的治理主体

### （三）易混易错点（重点辨析）

| 易混概念 | 区别要点 |
|---|---|
| **两个确立** vs **两个维护** | "两个确立"指确立习近平同志党中央的核心、全党的核心地位，确立习近平新时代中国特色社会主义思想的指导地位；"两个维护"中关键是维护党中央权威和集中统一领导，落到实处是维护党中央和各级党组织的领导地位。两者紧密联系但内涵不同，**应结合起来复习**，不能混为一谈。 |
| **维护党中央权威和集中统一领导** vs **党的民主集中制** | 二者完全一致、不矛盾，不要误以为"集中统一领导"和"民主集中制"是对立关系。 |
| **成果民主** vs **结果民主** | 我们强调的是"成果民主"，**不是**"结果民主"，两字之差但表述不同，考试中容易写错概念名称。 |
| **"一个变"与"两个没有变"** | 变的是社会主要矛盾；没有变的是：① 我国仍处于社会主义初级阶段的基本国情；② 我国仍是世界最大发展中国家的国际地位。注意区分"变"与"不变"分别对应的具体内容，不要张冠李戴。 |
| **"魂脉"与"根脉"** | 马克思主义基本原理是"魂脉"，中华优秀传统文化是"根脉"，二者结合构成"两个结合"，不要写反对应关系。 |
| **马克思主义中国化"新的飞跃"** | 课堂明确强调只提"新的飞跃"，**没有"第三次飞跃"的官方表述**，答题时不要自行加上"第三次"等限定词。 |
| **十四个坚持 / 十四个方略** | 同一内容的两种常见说法，答题时使用任一表述均可，但要知道两者指的是同一件事。 |

### 框架总览

```
导论：理论根基与历史定位
  └─ 第一章：新时代历史方位 ──┐
                              │
第二章：现代化道路与原则      │  
  └─ 第三章：党的领导制度      ├─ 中国特色社会主义的
  └─ 第四章：人民立场与共富    │   "为什么"与"怎么办"
  └─ 第五章：改革开放方法论 ──┘
                              
第六章：新发展理念与高质量发展
第七章：科技创新战略
第八章：全过程人民民主
                              ┐
第十章：文化自信与建党精神    │  治理体系与文明建设

第十二章：生态文明建设      ──┘
```

### 分章节复习要点、记忆要点与例题示范

#### 导论：理论根基
- **脉络**：时代背景（5方面）→ 两个结合（魂脉+根脉）→ 科学体系（十个明确/十四个坚持/十三个方面成就/六个必须坚持）→ 历史地位 → 两个确立
- **记忆要点**：用"背景—结合—体系—地位—确立"五个关键词串联导论逻辑链
- **例题解析**：
  - 问：如何理解"两个结合"？
  - 答题思路：先给出定义（马克思主义基本原理同中国具体实际相结合、同中华优秀传统文化相结合），再简述二者关系（"魂脉"与"根脉"的关系，二者统一于马克思主义中国化时代化的实践），最后可联系习近平新时代中国特色社会主义思想是"两个结合"的重大成果作总结。

#### 第一章：新时代的历史方位
- **脉络**：新时代的内涵 → 一个变两个没有变 → 道路决定性 → 两个奇迹 → 五位一体/四个全面 → 中国梦
- **重难点**：准确区分"变"与"不变"的具体指向；理解"道路问题是第一位问题"的逻辑（即一种主义能否真正解决国家面临的历史性课题）
- **记忆要点**："一变两不变"口诀化记忆——矛盾变了，国情阶段没变，国际地位没变
- **例题解析**：
  - 问：为什么说道路问题是关系党的事业兴衰成败的第一位的问题？
  - 答题思路：从"一个国家实行什么主义关键要看能否解决其历史性课题"切入，结合中国特色社会主义是历经千辛万苦、付出巨大代价取得的根本成就这一论断展开论述。

#### 第二章：人类文明新形态与中国式现代化
- **脉络**：创造人类文明新形态的三方面表现 → 推进中国式现代化的五大重大原则 → 五个必由之路（团结奋斗）
- **重难点**：五大原则与"四项基本原则"的结合记忆；"六对重大关系"的处理
- **记忆要点**：五大原则首字简记——党的领导、中国道路、人民中心、改革开放、斗争精神
- **例题解析**：
  - 问：中国式现代化是如何创造人类文明新形态的？
  - 答题思路：分三点展开——提供了全新的现代化路径，超越西方现代化模式；为广大发展中国家提供了多样化选择；推进中国式现代化必须牢牢把握五大重大原则。

#### 第三章：党的领导与制度优势
- **脉络**：最大国情 → 健全全面领导制度 → 党的自身优势（四力）→ 党的领导是全面系统整体的 → 两个维护
- **重难点**：理解"党的领导全面系统整体"不等于"事事都由党具体包办"；两个维护与民主集中制的一致性
- **记忆要点**：四力口诀——政治领导力（第一位能力）、思想引领力、群众组织力、社会号召力
- **例题解析**：
  - 问：为什么说中国共产党的领导是中国特色社会主义制度的最大优势？
  - 答题思路：先point出"中国最大的国情就是中国共产党的领导"，再从党的自身优势（四力）是制度优势的主要来源角度论述，最后联系党的领导得到人民拥护、由国家性质和政治体制决定等内容。

#### 第四章：人民立场与共同富裕
- **脉络**：民心是最大的政治 → 人民立场（根本政治立场+显著标志）→ 群众路线与调查研究 → 共同富裕（本质要求+重要特征+有底线渐进推进）
- **重难点**：共同富裕"有底线、渐进式推进"的辩证理解，避免理解为"齐头并进、同步达到"
- **记忆要点**："立场—路线—富裕"三段记忆法
- **例题解析**：
  - 问：如何理解共同富裕是中国特色社会主义的本质要求？
  - 答题思路：先明确定位（本质要求+中国式现代化重要特征），再阐述推进方式（有底线、渐进式推进，不是整齐划一同步实现）。

#### 第五章：全面深化改革开放
- **脉络**：深刻革命的判断 → 总目标（完善制度+治理现代化）→ 正确方法论（统筹推进、制度建设主线、经济体制改革重点、关键环节突破口）
- **重难点**：方法论中"主线—重点—突破口"三个层次不能混淆
- **记忆要点**："一场革命、一个目标、一套方法论"三句概括全章
- **例题解析**：
  - 问：全面深化改革开放为什么是一场深刻革命？
  - 答题思路：结合改革的广度、深度、影响力展开，说明其推动国家治理体系和治理能力现代化的根本性变革意义。

#### 第六章：新发展理念与高质量发展
- **脉络**：新发展理念五大内容（创新、协调、绿色、开放、共享）→ 基本经济制度新概括（所有制/分配制度/资源配置）→ 两个毫不动摇 → 新发展格局
- **重难点**：基本经济制度新概括的三个角度（所有制、分配制度、资源配置体制机制）需要分别记忆，不要笼统作答
- **记忆要点**："理念五字诀"（创新、协调、绿色、开放、共享）
- **例题解析**：
  - 问：新发展理念回答了哪些根本问题？
  - 答题思路：分别对应发展的目的、动力、方式、路径等问题逐一展开，体现理念与问题的对应关系。

#### 第七章：科技创新战略
- **脉络**：科技是第一生产力 → 人才是第一资源 → 创新是第一动力
- **记忆要点**："三个第一"顺口溜
- **例题解析**：
  - 问：如何理解"科技强则国家强"？
  - 答题思路：从"科技—人才—创新"三者关系展开，说明三者共同支撑国家强盛。

#### 第八章：社会主义民主政治
- **脉络**：人民民主是社会主义的生命 → 民主首先是国家制度 → 全过程人民民主四个相统一 → 成果民主≠结果民主
- **重难点**："成果民主"与"结果民主"的概念区分是易错点，需特别留意表述
- **例题解析**：
  - 问：全过程人民民主的本质属性体现在哪些方面？
  - 答题思路：先明确其是社会主义民主政治的本质属性，再展开"四个相统一"的具体内容。

#### 第十章：文化自信与复兴
- **脉络**：文化繁荣兴盛的必然要求 → 文化自信（更基础更广泛更深沉）→ 文化三个组成部分 → 意识形态（旗帜/道路/国家安全）→ 伟大建党精神
- **记忆要点**："文化—意识形态—建党精神"三点连线记忆
- **例题解析**：
  - 问：为什么说文化自信是更基础、更广泛、更深沉的力量？
  - 答题思路：结合"四个自信"整体框架，说明文化自信在其中的独特地位，再联系意识形态关乎旗帜、道路、国家安全的论断加以论证。

#### 第十一章：民生与社会治理
- **脉络**：人民生活幸福是国之大者 → 分配制度改革（促进公平）→ 市域社会治理现代化
- **记忆要点**："幸福—分配—治理"三段式
- **例题解析**：
  - 问：为什么要加快建设市域社会治理现代化？
  - 答题思路：突出市域是将风险矛盾化解在基层、解决在萌芽状态最直接、最有效率的治理主体这一核心论断。

#### 第十二章：生态文明建设
- **脉络**：摆在全局工作突出位置 → "绿水青山就是金山银山"理念指导实践
- **例题解析**：
  - 问：生态文明建设应摆在什么位置？
  - 答题思路：明确"全局工作的突出位置"这一表述，并联系绿色发展理念加以论述。
